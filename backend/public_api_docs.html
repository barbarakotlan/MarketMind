Private Beta

MarketMind Public API

Read-only MarketMind developer API for market data and research workflows. All requests use MarketMind developer API keys.

Available contracts: MarketMind Public API v1 and MarketMind Public API v2.

OpenAPI v1  |  OpenAPI v2

Authentication

Authenticate every request with a MarketMind developer API key:

Authorization: Bearer <marketmind_developer_api_key>

Keys are admin-issued for the private beta and are shown only once at creation time.

Rate Limits and Quotas

The private beta enforces per-minute, per-hour, and daily quotas per API key.

Responses may include:

  • X-RateLimit-Limit
  • X-RateLimit-Remaining
  • X-RateLimit-Reset
  • X-Public-API-Daily-Quota
  • X-Public-API-Daily-Remaining
  • X-Cache
  • X-Request-ID

Error Format

{
  "error": {
    "code": "invalid_ticker",
    "message": "Ticker is invalid.",
    "request_id": "0d69c0e9dcb147b0892bb9afdbd1eb8a"
  }
}

Common codes: invalid_api_key, api_key_revoked, api_key_expired, rate_limited, quota_exceeded, invalid_ticker, invalid_query, upstream_unavailable, not_found, internal_error.

Versioning and Changelog

v1 established the core read-only contract. v2 expands the read-only research surface while keeping the same auth, quota, and error model.
